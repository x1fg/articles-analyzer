Статья "Fortran2CPP: Automating Fortran-to-C++ Migration using LLMs via Multi-Turn Dialogue and Dual-Agent Integration" представляет новый подход к автоматизации миграции кода с языка Fortran на C++ с использованием крупных языковых моделей (LLMs). Авторы предлагают систему, которая интегрирует многотуровой диалог и взаимодействие двух агентов для более эффективного преобразования кода. Основное внимание уделяется сохранению логики исходного кода и его функциональности при одновременном обеспечении оптимизации и улучшения структуры кода на языке C++. Инновация заключается в использовании двух агентов: один агент отвечает за генерацию кода, а другой — за его верификацию и улучшение. Этот подход позволяет уменьшить количество ошибок и повысить качество получаемого C++ кода, облегчая задачу программистам и существенно снижая трудозатраты на процесс миграции.
