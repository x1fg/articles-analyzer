Статья "SafeSynthDP: Leveraging Large Language Models for Privacy-Preserving Synthetic Data Generation Using Differential Privacy" посвящена разработке метода генерации синтетических данных с использованием крупных языковых моделей (LLMs) и дифференциальной приватности. Авторы предлагают подход, который сочетает возможности LLMs в создании реалистичных данных с механизмами дифференциальной приватности для обеспечения конфиденциальности. Основная цель работы — предоставить способ генерирования данных, который защищает индивидуальную информацию, сохраняя при этом полезность данных для анализа и исследований. Метод SafeSynthDP демонстрирует, как можно эффективно балансировать между приватностью и качеством синтетических данных, что может быть полезно для различных приложений, требующих соблюдения строгих норм конфиденциальности.
